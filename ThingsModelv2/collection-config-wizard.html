工具介绍
了解采集配置工具的功能与流程，帮助您快速上手
采集配置工具 面向 EMS 与能源物联网场景，帮助用户为设备或站点创建标准化的采集物模型。通过引导式配置，完成设备信息录入、采集属性定义、告警规则配置，最终导出结构化配置文件。
设备信息配置

配置设备或站点的身份标识、通信接口、协议类型，以及采集指令。

属性配置

定义采集数据点，包括数据类型、寄存器映射、单位、精度等，支持 CSV 批量导入。

告警配置

定义告警规则，包括告警等级、触发条件、死区和触发时长，并关联到属性。

预览与导出

预览完整配置结果，导出为 JSON 或 YAML 格式文件，供系统导入使用。

配置流程
  1. 填写设备基本信息（名称、接口、协议等）
  2. 配置采集指令（Modbus 等协议的参数）
  3. 定义采集属性数据点及寄存器映射
  4. 配置告警规则和触发条件
  5. 预览并导出 JSON / YAML 配置文件
